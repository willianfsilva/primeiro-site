O que é HTML

HTML não é uma linguagem de programação; é uma linguagem de marcação, usada para definir a estrutura do seu conteúdo. HTML consiste de uma série de elementos, que você usa para delimitar ou agrupar diferentes partes do conteúdo para que ele apareça ou atue de determinada maneira.

Principais tags

  • Tags de cabeçalho
  • Tags de lista
  • Outras tags

O que são Tags

Uma tag HTML é um elemento utilizado para informar ao navegador a estrutura de um documento HTML. As tags são formadas por uma estrutura própria, iniciando com o sinal "menor que" e terminando com o sinal "maior que".

Tags de cabeçalho

Voltar

Os elementos de cabeçalhos permitem especificar que certas partes do seu conteúdo são títulos ou subtítulos. Da mesma forma que um livro tem o título principal e os capítulos possuem títulos e subtítulos, um documento HTML também tem. HTML contém 6 níveis de título, que vai do H1 ao H6.

Tags de lista

Voltar

Muito do conteúdo da web é de listas e o HTML tem elementos especiais para elas. Listas de marcação sempre consistem em pelo menos 2 elementos. Os tipos mais comuns de lista são ordenadas e não ordenadas:

  1. Listas não ordenadas são para listas onde a ordem dos itens não importa, como uma lista de compras, por exemplo. Essas são envolvidas em um elemento.
  2. Listas Ordenadas são para listas onde a ordem dos itens importa, como uma receita. Essas são envolvidas em um elemento.

Outras tags

Voltar
  • O elemento HTML "p" define um parágrafo.
  • A tag "mark" define o texto que deve ser marcado ou destacado.
  • A tag "small" define textos menores (como direitos autorais e outros comentários secundários).
  • A tag "i" exibe o texto em itálico
  • A tag "u" exibe o texto sublinhado
  • A tag "sub" exibe o texto subscrito.
  • A tag "sup" exibe o texto sobrescrito.
  • A tag "blockquote" defibe uma seção citada de outra fonte
  • A tag "abbr" define uma abreviatura
  • A tag "del" define um texto com uma parte excluída e uma nova parte inserida
  • A tag "strong" define um texto importante em um documento. O conteúdo é exibido em negrito
